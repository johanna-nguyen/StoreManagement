# Store Management System

A desktop application for managing products, inventory, import/export invoices, and users.  
The project is built using Java Swing, MySQL, and follows the MVC (Model–View–Controller) architecture.

# Features

- User authentication (Login / Role-based access)
- Product management
- Inventory (Stock) management
- Import and export invoices
- Quantity validation (prevent exporting more than available stock)
- Database connection via JDBC
- Clean MVC structure

---

# Technologies Used

- Java (JDK 24)
- Java Swing
- Maven
- MySQL 8.x
- JDBC
- MVC Architecture
